# 🎬 CineMatch

Quiz interativo de compatibilidade cinematográfica para casais!

## 🌐 Demo

Acesse: [https://mateusndjcinematech.github.io/Cinematch/](https://mateusndj.github.io/Cinematch/)

## 📖 Sobre

CineMatch é um quiz divertido onde casais descobrem o quanto combinam no cinema. Cada pessoa escolhe seus gêneros favoritos e responde perguntas sobre hábitos de assistir filmes, e o sistema calcula a compatibilidade usando o Índice de Jaccard.

### Funcionalidades

- ✨ Cálculo real de compatibilidade baseado em gêneros
- 🎭 7 perguntas personalizadas sobre hábitos cinematográficos
- 🎬 Recomendações de filmes via API do TMDB
- 💫 Animações suaves e design moderno
- 📱 Totalmente responsivo
- 🔒 Sem cadastro, sem dados salvos

## 🚀 Como usar

1. Acesse o link do GitHub Pages
2. Clique em "Começar o Quiz"
3. Digite os nomes do casal
4. Cada pessoa escolhe até 4 gêneros favoritos
5. Respondam as perguntas
6. Veja o resultado com porcentagem de compatibilidade e filmes recomendados!
