# CineMatch 🎬

## Como rodar

1. Compile todos os arquivos:
```
javac -d out src/*.java
```

2. Execute:
```
java -cp out Main
```

3. Acesse no browser: **http://localhost:8080**

## Observação
O projeto usa a API pública do TMDB. A chave padrão é de demonstração.
Para uso contínuo, crie sua chave gratuita em: https://www.themoviedb.org/settings/api
E substitua `API_KEY` em `MovieApiClient.java`.
